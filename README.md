# 📊 Global Superstore Sales Analysis using SQL

## 📖 Project Overview

This project presents an exploratory sales analysis of the **Global Superstore Sales** dataset using **SQLite**. The objective was to answer real-world business questions through SQL queries and gain hands-on experience with data analysis on a practical dataset.

The analysis focuses on sales performance, customer segments, product profitability, regional performance, and shipping costs. This project was completed as part of my **#100DaysOfCode** learning journey while studying SQL for Data Analytics.

---

## 🎯 Objectives

- Practice SQL on a real-world dataset.
- Strengthen analytical thinking through business-driven questions.
- Explore sales and profit trends across products, customers, and regions.

## 📊 Business Questions Answered

This project answers business questions such as:

- How many unique orders are there?
- How many customers have placed orders?
- What are the total sales generated?
- Which product category generated the highest sales?
- Which countries generated the highest sales?
- Which market generated the highest total profit?
- Which customer segment generated the highest total profit?
- Which sub-category generated the highest total sales?
- Which state generated the highest total profit?
- Which region has the highest average shipping cost?
- Which products generated sales above a specified threshold?
- Which are the top 10 most profitable products?
